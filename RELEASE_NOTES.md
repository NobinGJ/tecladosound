# v1.0 — TecladoSounds

## English

### What's new

- **CustomTkinter GUI** with dark mode (#1a1b26), logo in header & tray
- **17+ switch profiles** — MX Blue, Brown, Holy Panda, Alpaca, NK Cream, Logitech G915, and more
- **Real-time profile switching** — click a profile, sound changes instantly
- **System tray** — window minimizes to tray, right-click for Open / Settings / Close
- **Settings panel** inline (gear icon), auto-start with Windows, auto-start sound, default volume
- **Volume slider** in main window
- **Single toggle button** — Start (green) / Stop (red)
- **Compiled with PyInstaller** — single .exe, no dependencies required
- **Dark title bar** on settings popup (Windows DWM API)
- **Taskbar & window icon** now uses Logo.png

### Bug fixes

- Profile switching in real-time now works (motor restarts automatically)
- Tray icon settings popup has dark title bar
- Fixed file paths for PyInstaller frozen builds
- Removed broken tray profile switching (kept simple: tray only has Open/Settings/Close)

## Español

### Novedades

- **Interfaz gráfica** con CustomTkinter, modo oscuro (#1a1b26), logo en cabecera y bandeja
- **17+ perfiles de switches** — MX Blue, Brown, Holy Panda, Alpaca, NK Cream, Logitech G915 y más
- **Cambio de perfil en tiempo real** — haz clic en un perfil, el sonido cambia al instante
- **Bandeja del sistema** — la ventana se minimiza al cerrar, clic derecho para Abrir / Ajustes / Cerrar
- **Panel de ajustes** inline (icono engranaje), auto-inicio con Windows, auto-inicio de sonido, volumen predeterminado
- **Control de volumen** en la ventana principal
- **Botón único** — Iniciar (verde) / Detener (rojo)
- **Compilado con PyInstaller** — un solo .exe, sin dependencias
- **Barra de título oscura** en ventana de ajustes (API DWM de Windows)
- **Icono de ventana y barra de tareas** ahora usa Logo.png

### Corrección de errores

- El cambio de perfil en tiempo real ahora funciona (el motor se reinicia automáticamente)
- Ventana de ajustes desde la bandeja tiene barra de título oscura
- Rutas de archivo corregidas para compilación con PyInstaller
- Eliminado el cambio de perfil desde la bandeja (se mantiene simple: solo Abrir/Ajustes/Cerrar)
