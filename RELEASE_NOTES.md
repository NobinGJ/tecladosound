# v1.1 — TecladoSounds

## English

### What's new

- **Fixed key repeat** — holding a key no longer loops the sound (plays once per press)
- **Game support** — added raw input capture via Windows API so it works in games
- Key tracking unified between normal apps and games

### Bug fixes

- Key repeat when holding a key (now only plays on first press)
- Games/keyboard not detected (added raw input fallback)

---

## Español

### Novedades

- **Corregida repetición de tecla** — mantener presionada una tecla ya no repite el sonido en bucle (suena solo al presionar)
- **Soporte para juegos** — añadida captura de entrada raw mediante API de Windows para que funcione en juegos
- Seguimiento de teclas unificado entre aplicaciones normales y juegos

### Corrección de errores

- Repetición de sonido al mantener una tecla presionada (ahora suena solo la primera vez)
- Juegos no detectaban el teclado (añadido fallback de raw input)
