📖 Guía rápida

  1. Descarga y ejecuta TecladoSounds.exe
  2. Selecciona un perfil de la lista izquierda
  3. Presiona Iniciar
  4. Escribe en cualquier programa y escucha el sonido
  5. Cambia de perfil en cualquier momento

🔧 Compilar desde código

git clone https://github.com/TU_USUARIO/tecladosound.git
cd tecladosound
pip install -r requirements.txt
python tecla.pyw

Para compilar tu propio .exe:

pip install pyinstaller
pyinstaller --onefile --name TecladoSounds --noconsole --icon Logo.png --add-data "Logo.png;." --add-data "keyboardsounds\profiles;keyboardsounds\profiles" tecla.pyw
